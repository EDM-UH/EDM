Python 3.5+
	• Environments: 
		o Spyder
		o Jupyter Notebook
	• Libraries: 
		o Numpy for numerical computation and statistical analysis
		o Pandas for data preprocessing
		o Matplotlib, Seaborn, and Pandas for visualization
		o Scikit-learn for machine learning

Installation Guide:
	• Anaconda:
		o https://www.continuum.io/downloads
		o Troubleshoot:
			- The Anaconda Navigator may not work for some OS. 
				To access Jupyter Notebook open Anaconda Prompt 
				and type: jupyter notebook.  
	• Command Prompt to install Libraries:
		o pip install <library case-sensitive name>
